# mini_NPU

## 1. 프로젝트 개요
본 프로젝트는 3x3 필터와 입력 패턴 간의 MAC(Multiply-Accumulate) 연산을 수행하고 유사도를 판별하는 미니 NPU(Neural Processing Unit) 시뮬레이터입니다.

## 2. 주요 기능 및 동작 검증
프로젝트에서 요구하는 두 가지 모드의 정상 동작을 검증합니다.

### 모드 1: 단일 패턴 검사
* **기본 동작:** 3x3 필터 2개와 3x3 패턴을 입력받아 MAC 점수, 최종 판정 결과, 소요 시간(ms)을 출력합니다.
* **동점 처리:** 허용 오차 내에서 동점이 발생할 경우 "판정 불가"를 정상적으로 출력합니다.
* **오류 처리:** 입력된 행/열의 개수가 불일치하는 등 형식 오류 발생 시, 프로그램이 종료되지 않고 재입력을 유도합니다.

### 모드 2: 대량 데이터 로드 및 검사
* **데이터 로드:** JSON 형태의 데이터를 성공적으로 로드하고, 올바른 필터를 매칭하여 선택합니다.
* **라벨 정규화:** `expected` 값과 `filter` 키의 형식을 통일하는 표준 라벨 정규화를 수행합니다.
* **동점 처리:** 모드 2에서 동점이 발생할 경우 "UNDECIDED"를 출력하고, 전체 결과 집계에서 FAIL로 정상 처리합니다.
